Fork & Pull Request করার নিয়ম How to Fork & Create a Pull Request

অন্যের রিপোজিটরিতে কোড এডিট করে রিকোয়েস্ট পাঠানোর সম্পূর্ণ গাইড — ৪টি সহজ ধাপে। A complete guide to contributing code to someone else's repository — in 4 simple steps.

💡
যেহেতু আপনি ওনারের রিপোজিটরিতে সরাসরি push করার অনুমতি রাখেন না, তাই গিটহাবের স্ট্যান্ডার্ড নিয়ম হলো — প্রথমে Fork করে নিজের কপি বানান, এডিট করুন, তারপর Pull Request পাঠান। Since you don't have write access to the owner's repository, GitHub's standard approach is — first Fork it to your account, make your edits, then send a Pull Request.
1
গিটহাবে Fork করুন Fork the Repository on GitHub
  1. ব্রাউজারে ওনারের রিপোজিটরির পেজ-এ যান।
  2. পেজের ডানদিকে উপরে Fork বাটনটি খুঁজে ক্লিক করুন।
  3. গিটহাব আপনার নিজের অ্যাকাউন্টে ওই প্রজেক্টের একটি হুবহু কপি তৈরি করে দেবে।
  1. Go to the owner's repository page in your browser.
  2. Find the Fork button at the top right of the page and click it.
  3. GitHub will create an exact copy of the project under your own account.
✅ এখন আপনার অ্যাকাউন্টে একটি কপি রেডি! Your forked copy is now ready under your account!
2
আপনার Fork Clone করুন Clone Your Fork Locally

ওনারের রিপো নয়, আপনার কপিটি লোকাল মেশিনে ক্লোন করুন: Clone your fork (not the owner's) to your local machine:

bash
$ git clone https://github.com/YOUR-USERNAME/repo-name.git $ cd repo-name

⚠️ YOUR-USERNAME এর জায়গায় আপনার গিটহাব ইউজারনেম এবং repo-name এর জায়গায় আসল রিপোর নাম দিন। Replace YOUR-USERNAME with your GitHub username and repo-name with the actual repository name.

3
কোড এডিট করে Push করুন Edit Code & Push to Your Fork

কোড এডিট করার পর নিচের কমান্ডগুলো একে একে রান করুন: After editing the code, run these commands one by one:

bash
$ git add . $ git commit -m "Describe the edits you made" $ git push origin main
কমান্ডCommand কাজPurpose
git add . সব পরিবর্তন স্টেজ করেStages all changes
git commit -m মেসেজসহ পরিবর্তন সেভ করেSaves changes with a message
git push origin main আপনার গিটহাব অ্যাকাউন্টে পাঠায়Sends changes to your GitHub fork
4
Pull Request (PR) পাঠান Open a Pull Request
  1. ব্রাউজারে আপনার গিটহাব অ্যাকাউন্টের ওই রিপোর পেজে যান।
  2. Contribute বা Open Pull Request অপশনটি খুঁজে ক্লিক করুন।
  3. New pull request বাটনে ক্লিক করুন।
  4. একটি সুন্দর টাইটেল ও বিবরণ লিখে PR সাবমিট করুন।
  1. Go to your forked repository page on GitHub.
  2. Find and click the Contribute or Open Pull Request option.
  3. Click the New pull request button.
  4. Write a clear title and description of your changes and submit.
✅ ওনারের কাছে নোটিফিকেশন চলে যাবে! The repository owner will receive a notification!

✅ ওনার এরপর কী করবেন? What does the owner do next?

ওনারের কাছে PR নোটিফিকেশন পৌঁছাবে।The owner receives a PR notification.

তিনি আপনার কোড রিভিউ করবেন।They review your code changes.

সব ঠিক থাকলে Merge বাটনে ক্লিক করবেন।If everything looks good, they click the Merge button.

মার্জ হলেই আপনার কোড ওনারের আসল রিপোজিটরিতে যুক্ত হয়ে যাবে!Once merged, your code becomes part of the original repository!
